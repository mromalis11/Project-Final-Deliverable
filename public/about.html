Why we built GameStats Hub

Sports fans juggle multiple sites to track fixtures, standings, and favorite clubs. GameStats Hub condenses live scores, league tables, and personalized favorites into one reliable experience powered by modern web tooling.

View live scores Explore the dashboard

Tech Stack

Front-end is a lightweight, React-ready prototype using ES modules, Chart.js visualizations, and dayjs formatting. Backend runs on Express with Supabase for persistence and API-FOOTBALL as the live data source.

Express + Fetch Supabase Postgres Chart.js · dayjs

Pages

  • Home – live fixtures stream + standings snapshot
  • About – system goals, tech decisions, and instructions
  • Dashboard – search teams, view upcoming fixtures, and save favorites

Deployment Plan

Designed for Vercel: Express server deployed as Node runtime, public folder served statically, Supabase used via environment variables. A deployment checklist lives in the README developer manual.

Data: API-FOOTBALL v3 · Storage: Supabase · UI: custom CSS with purposeful gradients
